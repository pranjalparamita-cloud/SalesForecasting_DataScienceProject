📊 Walmart Sales Analysis Dashboard (Power BI)
📌 Project Overview

This project presents an interactive Sales Analysis Dashboard built using Power BI to analyze Walmart store sales data.
The dashboard provides insights into sales trends, store performance, holiday impacts, and economic indicators such as fuel price, unemployment rate, and CPI.

The goal of this project is to demonstrate data visualization, business intelligence, and analytical skills using Power BI.

🎯 Objectives

Analyze weekly sales trends across Walmart stores

Identify top performing stores

Compare holiday vs non-holiday sales

Examine how temperature, fuel price, CPI, and unemployment influence sales

Build an interactive dashboard for decision making

📂 Dataset

Dataset used: Walmart Store Sales Forecasting Dataset

Features
Column	Description
Store	Store ID
Date	Week of sales
Weekly_Sales	Total weekly sales
Holiday_Flag	Whether the week contains a holiday
Temperature	Average temperature
Fuel_Price	Fuel price in the region
CPI	Consumer Price Index
Unemployment	Unemployment rate
